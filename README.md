# CRUD Asp.Net

Web application, client registry using Asp.Net Framework 4.6.1 and SQL Server.

## Features

- Autentication
- Routes
- CRUD
- View Cshtml

## Requirements

- IIS
- Microsoft .NET Core SDK
- Microsoft SQL Server Express LocalDB

## Tecnologies

- C# 
- ASP.NET 4.6.1
- JQuery
- Bootstrap
- MSSQL
